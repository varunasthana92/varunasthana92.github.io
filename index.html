Projects


Object Detection: YOLOv3 TF2

YOLO is used for multiple object detection in a colored image. Version-3 supports detection of 80 different objects. In this project, YOLOv3 detection is implemented using pre-trained weights. YOLO utilizes anchor box priors to reduce false positive. For more robustness, 3 predictions are given across 3 scales. Non-maximum-suppression is used to filter the results.


Lane Detection

Objective of the project is to detect the current drive lane on the given data set (a video). Color segmentation done to generate a binary image. Sliding window used along the height of the image to detect all the points of left and right lanes.
